What is an opertator ?

An opertator is a method of packaging, deploying, and managing an application by extending the functionality of the Kubernetes API

What is the L5 operator &

The L5 operator is a demo operator that consumes a flask app and utilizes the Crunchy Postgres Operator.


Why was it created?

Following the Operator Framework, the L5 operator was created to demonstrate the capabilities of an operator from Level 1: Basic Installation to Level 5: Auto Pilot.

Operator Framework

Level 1: Basic Installation
Level 2: Seamless Upgrades
Level 3: Full Lifecycle
Level 4: Deep Insights
Level 5: Auto Pilot
